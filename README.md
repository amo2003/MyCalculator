# MyCalculator

A simple **React-based Calculator** application that performs basic arithmetic operations like addition, subtraction, multiplication, and division.

## Features

- Perform basic arithmetic calculations.
- Responsive and user-friendly interface.
- Styled using CSS for a clean layout.

## Technologies Used

- **React.js** – Frontend library
- **JavaScript (ES6)**
- **CSS** – Styling
- **Node.js & npm** – For project dependencies
